核心元素說明

DOCTYPE 聲明

  • 用途：聲明HTML版本
  • 語法：<!DOCTYPE html> (HTML5標準)

HTML root element

  • 作為所有其他元素的容器
  • lang屬性指定網頁語言
  • 示例：<html lang="en">

Head 區域

包含網頁的元信息：

  1. 字符編碼設置(防止亂碼)：<meta charset="UTF-8">
  2. 設定網頁大小(行動裝置): <meta name="viewport" content="width=device-width, initial-scale=1.0">
    • 作用: 讓網站可以適應手機, 平板, 電腦不同的螢幕大小
    • 解釋：
      • width=device-width → 網頁寬度跟螢幕寬度相同
      • initial-scale=1.0 → 預設縮放比例為 1(不放大、不縮小)
  3. 頁面標題：<title>網頁標題</title>

Body 區域

  • 包含所有可見的網頁內容
  • 如文字、圖片、連結等

List element

HTML提供兩種清單類型來組織內容：

  • 無序清單 <ul>: 用於項目順序無關緊要的場合

  • 有序清單 <ol>: 用於需要特定順序的項目

    Anchor element

    Attributes

    HTML attribute 可以為 element 提供額外的功能

    • 放在 starting tag 中
    • 在 tag name 之後, >角括號之前
      • ex. <a href=""></a> href 就是 attribute
    • 可以有多個 attribute, 用空格分開

    Anchor Element

    用來創建超連結

    1

    <a href="https://example.com">連結文字</a>

    Attribute type

    1. 特定屬性(Specific Attributes)：

      • 只能用於特定 element
      • 例如：anchor element 的href

    2. 全域屬性(Global Attributes)：

      • 可用於任何HTML元素
      • 例如：draggable、class、id等

    實際應用示例

    創建帶有超連結的有序列表：

    <li><a href="https://example2.com">網站二</a></li>
</ol>

    Image Element

    基本概念

    圖片元素(<img>)是HTML中用於顯示圖像的 void element, 不需要 closing tag, 因為它沒有 content

    基本語法

    1

    <img src="image-url" alt="image description" />

    必要屬性

    1. src：

      • 指定圖片的來源URL
      • 支持多種圖片格式(JPEG, PNG, GIF等)
      • 可以使用相對或絕對路徑

    2. alt:代文本屬性：

      • 提供圖片的文字描述
      • 用於無障礙訪問
      • 當圖片無法載入時顯示

    圖片格式支持

    支持的圖片格式包括：

    • JPEG：適合照片
    • PNG：適合需要透明背景的圖片
    • GIF：支持動畫效果
    • WebP：現代網頁優化格式

    無障礙性考慮

    替代文本(alt)的重要性：

    • 幫助使用螢幕閱讀器的用戶
    • 提升網站的可訪問性
    • 當圖片無法顯示時提供信息

    最佳實踐

    1. 始終添加有意義的alt屬性：
    1

    <img src="dog.jpg" alt="小狗在沙灘上挖沙" />
    1. 使用適當的圖片格式：
    1
2

    <!-- 動態圖片使用GIF -->
<img src="animation.gif" alt="動畫效果" />
    1. 確保圖片源路徑正確：
    1
2
3
4

    <!-- 相對路徑 -->
<img src="./images/photo.jpg" alt="照片" />
<!-- 絕對路徑 -->
<img src="https://example.com/image.jpg" alt="網路圖片" />
