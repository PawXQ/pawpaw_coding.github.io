Day 44 - Web Foundation - Intermediate to CSS

CSS Color Properties

basic

  1. color: 設定文字顏色
  2. background-color: 設定背景顏色
1
2
3
4

h1 {
  color: blue;
  background-color: red;
}

顏色表示方法

Named Colors

直接使用已經定義好的顏色名稱：

1
2
3
4

h1 {
  color: cornflowerblue;
  background-color: antiquewhite;
}

Hex Codes(十六進制)

使用 # 後接六位數字/字母：

1
2
3
4

h1 {
  color: #5D3891;
  background-color: #F99417;
}

顏色值格式

  1. Named Colors

    • 預定義顏色名稱
    • 例如: red, blue, green, whitesmoke

  2. Hex Codes

    • 格式: #RRGGBB
    • 每兩位代表 RGB 色值
    • 範圍: 00-FF

CSS Font Properties

font-size

static unit

1
2
3
4

h1 {
  font-size: 20px;  /* pixel(像素) */
  font-size: 12pt;  /* point(點數) */
}

relate unit

1
2
3
4

h1 {
  font-size: 2em;   /* 相對於父元素(相對於上層) */
  font-size: 2rem;  /* 相對於根元素(一般是 html level) */
}

單位說明：

  1. px: 1/96 英寸
  2. pt: 1/72 英寸
  3. em: 相對於父元素的字體大小
  4. rem: 相對於根元素的字體大小

font-weight

1
2
3
4
5

p {
  font-weight: normal;  /* 正常 */
  font-weight: bold;    /* 粗體 */
  font-weight: 700;     /* 數值(100-900) */
}

font-family

指定字體時, 建議提供備選字體以確保顯示效果

1
2
3
4

body {
  font-family: Helvetica, sans-serif;
  font-family: "Times New Roman", serif;
}

字體類型：

  1. serif: 襯線字體
  2. sans-serif: 無襯線字體
  3. monospace: 等寬字體
  4. cursive: 草書字體
  5. fantasy: 裝飾字體

Google Fonts 使用

不怕選定的 font user 會沒有

1

<link href="fonts.googleapis.com/css2?family=Font+Name" rel="stylesheet">

1
2
3

h1 {
  font-family: "Font Name", backup-font;
}

text-align

1
2
3
4
5
6
7

p {
  text-align: left;     /* 靠左 */
  text-align: center;   /* 置中 */
  text-align: right;    /* 靠右 */
  text-align: start;    /* 開始位置 */
  text-align: end;      /* 結束位置 */
}

使用建議

  1. 字體大小

    • 優先使用 rem 實現響應式設計
    • 避免過多層級的 em 嵌套

  2. 字體系列

    • 始終提供備用字體
    • 多字詞字體名需要使用引號
    • 考慮使用 Google Fonts 等網路字體

CSS Box Model

在 HTML 中, 每個 element 都視為一個 box, 空間有內到外

  • Content(content)
  • Padding(內邊距)
  • Border(邊框)
  • Margin(外邊距)

Width & Height

決定 element 基本的尺寸

1
2
3
4

element {
  width: 300px;  /* 或使用百分比 width: 100% */
  height: 200px;
}

Border

element 的外圍邊框

1
2
3
4
5

element {
  border: 10px solid black;    /* 簡寫方式 */
  border-width: 20px 10px;     /* 上下 左右 */
  border-top: 0px;             /* 單邊 */
}

Padding(內邊距)

決定 content 和 border 的距離

1
2
3
4
5

element {
  padding: 20px;               /* 四周相同 */
  padding: 10px 20px;          /* 上下 左右 */
  padding: 0 10px 20px 30px;   /* 上 右 下 左 */
}

Margin(外邊距)

控制 element 和其他 element 之間的距離

1
2
3
4
5

element {
  margin: 10px;                /* 四周相同 */
  margin: 20px 40px;           /* 上下 左右 */
  margin-top: 100px;           /* 單邊設定*/
}

Div container

<div> element 用來創建 container, 可以將多個 element 當作一個整體

1
2
3
4

<div>
  <p>content</p>
  <img src="image.jpg">
</div>

IMG

注意事项

  1. Border 會向外延伸, 不影響 content 區域大小
  2. Margin 可能會發生疊加
  3. Padding 會增加 element 的實際顯示大小
  4. Box Model 各屬性可以分別指定四個方向的值
